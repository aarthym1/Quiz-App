# Quiz App

A simple and interactive Quiz Application built using **React**.  
The app allows users to answer multiple-choice questions, highlights selected options, supports hover effects, and displays the final score.

## Features

- Multiple-choice quiz questions
- Real-time score calculation
- Selected option highlighting
- Hover effects on options
- Prevents multiple selections
- Clean and responsive UI
- Final result display
